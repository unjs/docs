# Getting Started

install packageName from npm

You can install packageName from [npm](https://npmjs.com/packageName):

::code-group

```sh [auto]
npx nypm i packageName
```

```sh [npm]
npm install packageName
```

```sh [Yarn]
yarn add packageName
```

```sh [pnpm]
pnpm add packageName
```

```sh [bun]
bun add packageName
```

::
